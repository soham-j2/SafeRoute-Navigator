
# SafeRoute Navigator
Open-source disaster response and safe route planning platform.  
Built for PVG HackHub 2025.  

## Features
- Hazard reporting (floods, accidents, roadblocks)
- Safe route planner (avoids hazards, shows safety score)
- SOS to trusted contacts
- 100% open source
